# Excel 智能分析与可视化平台

一个面向 Excel/CSV 的数据分析、仪表盘与可视化大屏平台。

## 核心目标

- 上传常规 Excel/CSV，自动识别 Sheet、表头与字段类型。
- 用户可修正字段类型，进行清洗、筛选、聚合与计算。
- 分析结果可保存并复用到仪表盘和可视化大屏。
- 保存数据绑定、分析逻辑、图表样式和页面布局，而不是静态截图。
- 替换同结构数据后，原有分析、仪表盘和大屏自动刷新。
- 字段新增、改名或删除时，提供差异检测与重新映射。

## MVP 技术方向

- Web：React + TypeScript
- 图表：Apache ECharts
- API：FastAPI
- 元数据与配置：PostgreSQL
- 文件与版本：S3 兼容对象存储
- 异步任务：Redis + Worker
- 分析执行：DuckDB / Polars

## 第一阶段目标

完成以下可演示闭环：

1. 上传 Excel/CSV；
2. 选择 Sheet 和表头；
3. 自动识别并修正字段类型；
4. 拖拽字段生成分析图表；
5. 保存分析并添加到仪表盘或大屏；
6. 保存布局与样式；
7. 上传新版数据后自动刷新图表；
8. 字段结构变化时提示重新映射。

## 开发约定
