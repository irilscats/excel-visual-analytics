# MVP 产品范围

## 产品定位

平台以 Excel/CSV 为主要数据入口，同时提供数据分析工作台、业务仪表盘和自由可视化大屏。

## 核心用户流程

1. 新建项目并上传 Excel/CSV；
2. 选择 Sheet、表头行与有效数据范围；
3. 系统推断字段类型与业务角色；
4. 用户修正字段、清洗数据并创建计算字段；
5. 在分析工作台拖拽维度和指标生成图表；
6. 保存分析对象并添加到仪表盘或大屏；
7. 保存布局、样式、筛选和联动；
8. 上传新版文件后自动刷新全部有效图表。

## MVP 必须包含

### 数据导入

- `.xlsx`、`.xls`、`.csv` 上传；
- Sheet 列表与预览；
- 表头行、数据起始行设置；
- 空行、空列和基础格式异常提示。

### 字段模型

- 文本、整数、小数、金额、百分比、日期、日期时间、布尔、分类、标识字段；
- 维度、指标、时间、状态、地理、描述等业务角色；
- 自动推断与人工覆盖；
- 稳定字段 ID 与表头映射。

### 分析工作台

- 维度、指标、聚合、筛选、排序与 Top N；
- 求和、平均、最大、最小、计数、去重计数；
- 指标卡、柱状图、条形图、折线图、饼图、散点图、数据表；
- 保存并再次编辑分析对象。

### 仪表盘与大屏

- 添加已保存分析；
- 拖拽、缩放、复制、删除和锁定组件；
- 保存画布尺寸、组件位置和图表样式；
- 全局筛选和基础图表联动；
- 预览与全屏展示。

### 数据刷新

- 以新文件创建数据版本；
- 同结构字段自动映射；
- 新增、改名、删除、类型变化检测；
- 仅在验证成功后切换活动版本；
- 失败时保留上一成功版本；
- 支持版本回滚。

## 非 MVP 范围

- 数据库和实时 API 数据源；
- 多表复杂关联；
- 多租户与精细权限；
- AI 自然语言分析；
- 高级地图和动画组件；
- 移动端原生应用。

## MVP 成功标准

使用一份包含日期、分类和数值字段的 Excel 完成从上传、分析、保存大屏到替换新版数据并自动刷新的完整演示，且无需重新布置组件。
